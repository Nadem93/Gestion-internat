🔐 Les données de santé sont stockées exclusivement dans le localStorage du navigateur. Aucune transmission externe. Pour une conformité HDS complète, déployez l'application sur un serveur certifié HDS (hébergement professionnel de données de santé).
Sauvegarde & restauration
💡 Sauvegardez régulièrement vos données. En cas de réinitialisation du navigateur, elles seraient perdues. Le « backup complet » couvre désormais tous les modules (résidents, équipe, planning, budget, paie, entretiens, annuaire, documentation, etc.).
